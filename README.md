# ❤️ Gesture Controlled Heart using Hand Tracking

A creative web project where a fist gesture forms a heart using animated particles.

## ✨ Features
- Real-time hand tracking with MediaPipe
- Gesture-based interaction
- Smooth canvas animations
- Lightweight and optimized

## Tech Stack
- HTML
- CSS
- JavaScript
- MediaPipe Hands

## How to Run
1. Clone the repository
2. Open `index.html` in a browser
3. Allow camera access
